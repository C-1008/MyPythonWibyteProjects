## Short Code

Use the function short_form to create a short code. Be creative but not too creative.

```python
sc = short_form(idx, a few words about yourself, prefix = "", suffix = "")
```
For example,

```python
sc = short_form(0, "Smart", "intelligent", prefix = "Mr", suffix = "Great")
```

Note, you don't _actually_ have to be reading the words you are using, it doesn't matter.
